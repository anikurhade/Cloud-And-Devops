   ls - List directory contents

Example: ls /home - This command will list the contents of the home directory.

	•     cd - Change directory

 Example: cd /var/log - This command will change the current directory to /var/log.

	•     pwd - Print working directory

 Example: pwd - This command will print the current working directory.

	•     mkdir - Make directory

Example: mkdir new_dir - This command will create a new directory called new_dir.

	•     rmdir - Remove directory

Example: rmdir old_dir - This command will remove the directory called old_dir.

	•     cp - Copy files or directories

Example: cp file1.txt file2.txt - This command will copy file1.txt to file2.txt.

	•     mv - Move or rename files or directories

 Example: mv file1.txt new_dir - This command will move file1.txt to the new_dir directory.

	•     rm - Remove files or directories

  Example: rm file1.txt - This command will remove file1.txt.

	•     cat - Concatenate and display files

   Example: cat file1.txt - This command will display the contents of file1.txt.

	•     grep - Search for a pattern in a file

  Example: grep "pattern" file1.txt - This command will search for the pattern in file1.txt.

	•     tar - Create or extract archives
    Example: tar -cvf archive.tar file1.txt - This command will create an archive called archive.tar containing file1.txt.

	•     chmod - Change file or directory permissions
    Example: chmod 755 file1.txt - This command will set the permissions of file1.txt to rwxr-xr-x.
